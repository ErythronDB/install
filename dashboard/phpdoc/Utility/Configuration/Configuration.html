Class Details



[line 19]

This class holds the application's configuration key/value pairs.

Full application configuration is generated from several sources,     including the main application config.ini and an optional config.local.ini     which can override values from the former.
This also includes the http port value parsed from a workers.properties     file (or equivalent) and the name of the Tomcat instance retreived from     the Apache server environment.
