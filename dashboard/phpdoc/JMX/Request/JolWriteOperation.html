Class Details

[line 41]
A single Jolokia WRITE operation for adding to a JolRequest.

Supply the full POST data payload as an array to the constructor. The array should json_encode() to a valid json POST request.

For example, use 

   new JolReadOperation(array(
       'mbean' => 'java.lang:type=ClassLoading',
       'attribute' => 'Verbose',
       'value' => 'true',
       'path' => null,
       ));
to produce the HTTP POST json for Jolokia 
      {"type":"write",
       "mbean":"java.lang:type=ClassLoading",
       "attribute":"Verbose",
       "value":"true",
       "path":null}

The mbean, attritribute and value properties must be set.

After instantiating the Operation object, add it to a JolRequest 

 $op = new JolWriteOperation($array);
 $req = new JolRequest;
 $req->add_operation($op);
