Class Details

[line 42]
A single Jolokia EXEC operation for adding to a JolRequest.

Supply the full POST data payload as an array to the constructor. The array should json_encode() to a valid json POST request.

Mulit-valued arguments should be passed in as an array() such that the json produced by json_encode($array) is a valid input to Jolokia.

For example, use 

   new JolExecOperation(array(
       'mbean' => 'java.util.logging:type=Logging',
       'operation' => 'setLoggerLevel',
       'arguments' => array('global', 'INFO'),
       ));
to produce the Jolokia json input 
   {"type":"exec",
    "mbean":"java.util.logging:type=Logging",
    "operation":"setLoggerLevel",
    "arguments":["global","INFO"]}

After instantiating the Operation object, add it to a JolRequest 

 $op = new JolExecOperation($array);
 $req = new JolRequest;
 $req->add_operation($op);

$mbean and $operation properties must be set
