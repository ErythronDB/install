method request [line 41]

	

    

    

		string request(
)
    

    


	
		Returns the json request handled by Jolokia and leading to  this response result.

This can differ from the original input json.  For example, unknown attributes are discarded and an empty  request is treated by Jolokia as a request for '{"type":"version"}'.
