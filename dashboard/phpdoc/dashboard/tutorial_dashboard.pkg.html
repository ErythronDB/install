/dashboard

Introduction

/dashboard is a web interface that provides state information for a EuPathDB website, including data from Apache HTTP Server, Tomcat and the WDK.

The Tomcat and WDK information is obtained from JMX mbeans via Jolokia's JMX-HTTP bridge. Apache server information is obtained directly from PHP's libraries.
